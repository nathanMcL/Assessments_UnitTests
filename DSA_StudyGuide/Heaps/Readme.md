## Heap

- **Big`O` Runtime**:  
  - **Access (Peek)**: `O(1)`  
  - **Insertion**: `O(log n)`  
  - **Deletion**: `O(log n)`  

- **Use Cases**:  
  - Implementing priority queues.  
  - Finding the smallest or largest element efficiently (min-heap or max-heap).  
  - Scheduling tasks where priority determines execution order.
